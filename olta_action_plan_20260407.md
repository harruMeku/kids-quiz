### 1-4. うまくいっていないこと

4. **3人で42行は物理的に手薄** → 既存パートナーフォロー不足（杉山さん自己評価）
5. **銀行員のOLTA推奨に対する自信不足**（手数料率不透明、審査厳しい印象）
6. **前月比78%**: 3月→4月の減速トレンド
7. **旧ティア分類（GTV×銀行規模）が機能していない** → 戦略的リソース配分の根拠が弱い

## 2. コア戦略（The One-Line Thesis）

> **「42行を3人で均等にカバーするのではなく、"業績評価×規模"の新軸で銀行を再分類し、上位10行に戦力の70%を集中して月間GTV+3〜5億を構造的に生み出す」**

### なぜこの戦略か

- **事実1**: 業績評価にOLTAが入っている行は高パフォーマンス（筑邦・豊和・千葉信金・肥後で確認済み）
- **事実2**: 旧分類（GTV×銀行規模）は相関なし → 別の変数が成果を決めている
- **事実3**: RS率とGTVも無相関（群馬RS15%でトップ、RS20%でも低GTV行多数）
- **結論**: GTV成長の最大レバーは「銀行内部の仕組み化（業績評価・研修・推奨フロー）」であり、これを3人のリソースで最大化するには選択と集中が不可欠

## 5. リソース配分: 42行を3人でどう分けるか

### 基本原則

**「人ではなくティアで管理する」**

42行を個人に均等割り（14行/人）するのではなく、ティア別のアクション内容で分担する。

### 具体的分担

```
【尾崎さん】 担当: S/Aティア行への現場活動
  - 帯同訪問: 月10-12回（S行に月2回、A行に月1回）
  - 対象: 群馬、筑邦、三十三、西日本シティ、肥後、豊和、千葉信金 + A行3-5行
  - 合併行への訪問も優先的に担当
  - 移動効率: 地域クラスタリングで訪問ルート最適化

【杉山さん】 担当: データ分析 + 全行パートナーサクセス
  - 行別分析レポート: 月次でS/A行、四半期でB/C行
  - ファネル分析: 成約率改善の継続的モニタリング
  - 提案書作成: 尾崎さんの訪問に合わせた行別カスタマイズ資料
  - B/Cティア行へのオンライン定例MTG運営サポート

【須藤】 担当: チームハブ + B/Cティア行のリモート管理
  - B/Cティア行（25-30行）のメール・電話での定期コンタクト
  - 週次チームMTGのファシリテーション
  - 尾崎さんの訪問スケジュール最適化
  - 杉山さんの分析結果のアクション化（提案書構成、社内調整）
  - 齊藤さんへの週次レポーティング
```

### なぜこの配分が機能するか

1. **尾崎さんの強み（対面力）を最もGTVインパクトの大きいS/A行に集中**
2. **杉山さんの強み（データ分析）を全行に横展開可能な形で活用**
3. **須藤の調整力で、尾崎・杉山の生産性を倍増させるハブ機能を実現**
4. **B/Cティア行はリモートで最小コスト運用し、S/A行にリソースを回す**

```
  ※ 正確にはGTVのうち金融機関チャネル約85% = 193億 × 7.5% = 14.5億 × 17% = 2.5億
- 保険料:       -2.4億円（227億×1.05%）
- デフォルト:    -2.3億円（227億×1%）
- オペコスト:    推定別途
= 粗利:         約9.8億円
```

**注**: 利益目標4.5億の達成可能性はオペコスト・資金調達コスト次第。GTVベースでは227億円が射程圏内。

## 8. なぜこのプランは機能するか

### 論理構造

```
前提1: 業績評価にOLTAが入っている行はGTVが高い（4行で確認済み）
前提2: 旧分類（GTV×規模）は機能しない → 新しい分類軸が必要
前提3: 3人で42行を均等にカバーするのは物理的に不可能
前提4: チームメンバーにはそれぞれ明確な強みがある

→ 結論: 新分類で選択と集中し、各メンバーの強みを最適配置すれば、
        同じ3人のリソースで1.5〜2倍のGTVインパクトを生み出せる
```

### 「がんばろう」ではない構造的根拠

1. **選択と集中**: 42行均等（現状）→ S/A行に70%集中（変更後）で、同じ労力でGTVインパクト2倍
2. **再現可能モデル**: 群馬の成功要因を分解し、他行に移植する。属人的ではなく構造的
3. **ファネル改善**: 申込件数を増やさなくても、成約率5pt改善で月+1.5〜2億の構造効果
4. **仕組み化**: B/Cティア行はリモート運用で月2-3日。残りの17-18日をS/A行に投入
5. **測定可能**: 全アクションにKPIと期限を設定。月次でPDCAを回す

### 各メンバーにとっての「やりがい」

- **尾崎さん**: 帯同訪問のエースとして、最も重要な行を任される。成果が最もダイレクトに見える役割。「自分の訪問がGTVを動かしている」実感
- **杉山さん**: データ分析が戦略の根幹を支える。「整理係」から「戦略の設計者」へ。業績評価分析という新しい切り口の発見者としてのクレジット
- **須藤**: チームのハブとして全体を機能させる。B/Cティア行の管理という明確な担当領域。24年の営業経験を活かした調整力で貢献
